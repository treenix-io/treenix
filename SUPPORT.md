# Support

## Questions

Use GitHub Discussions or community chat for usage questions, architecture questions, and help building mods.

- Docs: [docs/](./docs)
- Discord: https://discord.gg/peX8CwHQPz
- Telegram: https://t.me/treenix_io

## Bugs

Open a GitHub issue with:

- Treenix version or commit.
- Node.js version.
- Operating system.
- Minimal reproduction.
- Expected behavior and actual behavior.

## Feature Requests

Open a GitHub issue or discussion. Describe the use case, not only the proposed API.

## Security

Do not open public issues for vulnerabilities. See [SECURITY.md](./SECURITY.md).
