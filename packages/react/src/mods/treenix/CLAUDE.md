# treenix

Platform system mod. Actions for the `/sys` node: type discovery (`catalog`, `search_types`, `describe_type`), JSX view compilation, prefab deployment. `t.mod` module registry.

## Types
- `treenix.system` — actions: catalog, search_types, describe_type, compile_view, deploy_prefab
- `t.mod` — `name`, `state` (discovered/loading/loaded/failed/disabled)
